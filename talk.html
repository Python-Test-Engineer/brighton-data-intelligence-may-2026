Mental model shift

180 Degrees

Like an upside-down mouse — the same movements (left, right, up, down) but in the opposite direction to what we're used to.

3 Key Shifts in Agentic AI

  1. Client-side creation of endpoints (APIs) rather than server-side prebuilt endpoints.
  2. Use of Natural / Human Language (in my case English) to create the code logic.
  3. Autonomy — the LLM directs the flow of the application.
Pattern

Input → Function → Output

For this talk I'll use function in the mathematical sense:

input → function(input) → output → function(output) → output2

The function might be a Python function, a class, or another Agent that accepts the previous output as its input.

Under the hood

It's Just a REST API

Authentication: pass a token in the headers. One endpoint. Text in, text out.

model = "gpt-3.5-turbo"
model_endpoint = "https://api.openai.com/v1/chat/completions"

headers = {
    "Content-Type": "application/json",
    "Authorization": f"Bearer {api_key}",
}

payload = {
   "model": model,
   "messages": [
       {"role": "system", "content": system_prompt},
       {"role": "user",   "content": user_prompt},
   ],
   "stream": False,        # can stream response
   "temperature": temperature,   # degree of randomness
}

response = requests.post(
   url=model_endpoint,
   headers=headers,
   data=json.dumps(payload)
).json()

Key takeaways: There is only one endpoint. We don't use different endpoints for different tasks — we create our custom logic through prompt engineering. The request is text; the response is text.

Configuration

Temperature

Controls the degree of variation / randomness in the model's response.

Low Temperature

The bag is full of mostly blue marbles, with a few red and green. Low temperature means you're very likely to pull a blue marble, but might occasionally get a red or green.

High Temperature

The bag is filled with a mix of colours and all are equally likely. High temperature means you're equally likely to pull any colour, including uncommon ones.

Putting it together

The Essence

We generate a response with our first query using a system prompt to create code, then pass the output into another function that acts as a reviewer to produce the next version.

ReAct Pattern

Like onboarding a new employee: they receive a handbook of what the job involves and how to do it — that's our system prompt.
